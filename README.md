# jsai-report-template

Latex template for JSAI Conference


### 1. Prerequisities

```bash
# only for ubuntu 12.04
$ sudo apt-add-repository ppa:texlive-backports/ppa
$ sudo apt-get update
```

### 3. Make pdf

